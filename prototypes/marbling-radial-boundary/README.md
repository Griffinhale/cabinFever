# Marbling Radial Boundary Growth Prototype

## Purpose

Standalone p5.js prototype for testing sampled radial-ray boundary growth as a baseline/control architecture for cabinFever marbling.

## Run

```bash
cd /workspace/projects/Griffinhale_git/worktrees/cabinFever-marbling-radial/prototypes/marbling-radial-boundary
python3 -m http.server 8123
```

Open `http://localhost:8123`.

## Current state

This checkpoint contains the shell: p5 CDN, full-screen cream paper surface, subtle grain, resize handling, and tiny hints.

## Controls

- R or Escape: reset/rebuild the paper surface

More controls and behavior are added in later checkpoints.
