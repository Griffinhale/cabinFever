# Marbling Metaball Pools Prototype

Standalone p5/HTML shell for the metaballs / implicit pigment pools marbling prototype.

## Run

```bash
cd /workspace/projects/Griffinhale_git/worktrees/cabinFever-marbling-metaballs/prototypes/marbling-metaball-pools
python3 -m http.server 8123
```

Open `http://localhost:8123`.

## Planned controls

- Press/tap: create pigment.
- Hold: increase paint amount.
- P: cycle palette.
- R: reset.
